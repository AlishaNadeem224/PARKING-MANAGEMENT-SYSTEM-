# PARKING-MANAGEMENT-SYSTEM-
Parking Lot Management System is a C-based console application that manages parking operations with user registration, login, vehicle entry/exit, parking slot allocation, and fee calculation. The system uses file handling to store user and parking data and includes both admin and user modules.
The Parking Lot Management System is a console-based application developed in C that automates the process of managing a parking lot. The system allows users to register, log in, park vehicles, remove vehicles, and check parking availability. It also includes an admin panel that allows the administrator to manage parking slots, view records, and monitor system data.

The project uses file handling to store user data and parking records, ensuring that all data is saved even after the program is closed. The system is menu-driven and user-friendly, making it easy to manage parking operations efficiently.

This project demonstrates the use of core C programming concepts such as structures, functions, arrays, loops, conditional statements, and file handling.

How to Run the Project
  Open the project in any C compiler (Code::Blocks / Dev C++ / GCC).
  Compile all the source files.
  Run the program.
  Choose from Login, Register, or Exit.
  After login, access Admin or User menu.
